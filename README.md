Diski is a CLI tool that displays disk info of a system's storage devices 
in a nice table. 

```
┌──────────────┬────────────┬────────┐
│ Device       ┆ Filesystem ┆ Size   │
╞══════════════╪════════════╪════════╡
│ /dev/zram0   ┆ swap       ┆ 8G     │
├╌╌╌╌╌╌╌╌╌╌╌╌╌╌┼╌╌╌╌╌╌╌╌╌╌╌╌┼╌╌╌╌╌╌╌╌┤
│ /dev/nvme0n1 ┆ Unknown    ┆ 248.8G │
└──────────────┴────────────┴────────┘
```
